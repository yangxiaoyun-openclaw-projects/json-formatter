JSON格式化工具 - API文档

REST API 文档

为开发者提供的JSON格式化、验证、转换API接口

📋 API 概览

BASE URL http://[your-domain]:5000
所有API端点均支持跨域请求（CORS enabled），返回JSON格式数据。

🔤 格式化API

POST /api/format
格式化JSON数据，添加缩进和换行。

请求参数

参数 类型 必填 描述
json string 是 要格式化的JSON字符串
indent number 否 缩进空格数（默认：2）

cURL 示例

curl -X POST http://localhost:5000/api/format \ -H "Content-Type: application/json" \ -d '{"json": "{\"name\":\"John\",\"age\":30,\"city\":\"New York\"}", "indent": 4}'

响应示例

{ "formatted": "{\n \"name\": \"John\",\n \"age\": 30,\n \"city\": \"New York\"\n}", "size": 65, "valid": true }

🗜️ 压缩API

POST /api/compress
压缩/最小化JSON数据，移除所有不必要的空格和换行。

请求参数

参数 类型 必填 描述
json string 是 要压缩的JSON字符串

cURL 示例

curl -X POST http://localhost:5000/api/compress \ -H "Content-Type: application/json" \ -d '{"json": "{\\n \\"name\\": \\"John\\",\\n \\"age\\": 30,\\n \\"city\\": \\"New York\\"\\n}"}'

响应示例

{ "compressed": "{\"name\":\"John\",\"age\":30,\"city\":\"New York\"}", "reduction": 12, "size": 41, "valid": true }

✅ 验证API

POST /api/validate
验证JSON语法并提供修复建议。

请求参数

参数 类型 必填 描述
json string 是 要验证的JSON字符串

cURL 示例

curl -X POST http://localhost:5000/api/validate \ -H "Content-Type: application/json" \ -d '{"json": "{\\"name\\": \\"John\\", \\"age\\": 30, \\"city\\": \\"New York\\""}'

成功响应

{ "issues": [], "size": 47, "structure": "object", "valid": true }

错误响应

{ "error": "Expecting ',' delimiter: line 1 column 44 (char 43)", "suggestions": ["Check for missing commas between items"], "valid": false }

🔄 转换API

POST /api/convert
将JSON转换为其他格式（CSV、YAML、XML）。

请求参数

参数 类型 必填 描述
json string 是 要转换的JSON字符串
format string 否 目标格式：csv、yaml、xml（默认：csv）

cURL 示例

# 转换为CSV curl -X POST http://localhost:5000/api/convert \ -H "Content-Type: application/json" \ -d '{"json": "[{\\"name\\": \\"John\\", \\"age\\": 30}, {\\"name\\": \\"Jane\\", \\"age\\": 25}]", "format": "csv"}' # 转换为YAML curl -X POST http://localhost:5000/api/convert \ -H "Content-Type: application/json" \ -d '{"json": "{\\"user\\": {\\"name\\": \\"John\\", \\"age\\": 30}}", "format": "yaml"}' # 转换为XML curl -X POST http://localhost:5000/api/convert \ -H "Content-Type: application/json" \ -d '{"json": "{\\"person\\": {\\"name\\": \\"John\\", \\"age\\": 30}}", "format": "xml"}'

📦 批量处理API

POST /api/batch
批量处理多个JSON文件（格式化、压缩）。

请求参数（multipart/form-data）

参数 类型 必填 描述
files[] file[] 是 多个JSON文件
operation string 否 操作类型：format、compress（默认：format）
format string 否 输出格式：formatted、compressed（默认：formatted）

cURL 示例

curl -X POST http://localhost:5000/api/batch \ -F "files[]=@data1.json" \ -F "files[]=@data2.json" \ -F "operation=format" \ -F "format=formatted"
POST /api/download_batch
下载批量处理结果为ZIP文件。

请求参数

参数 类型 必填 描述
files array 是 文件数组，每个元素包含filename和content

请求示例

{ "files": [ { "filename": "data1_formatted.json", "content": "{\\n \\"name\\": \\"John\\"\\n}" }, { "filename": "data2_formatted.json", "content": "{\\n \\"name\\": \\"Jane\\"\\n}" } ] }

📖 错误码

状态码 描述
200 成功
400 请求参数错误或JSON格式无效
500 服务器内部错误
